Replicating the paper from https://github.com/erwincoumans/motion_imitation in Nvidia Isaac Gym.

# Running Scripts

Install Isaac in the project
``` 
 doskey PYTHON_PATH=C:\Users\nuggl\AppData\Local\ov\pkg\isaac_sim-2023.1.1\python.bat $*
 PYTHON_PATH -m pip install -e .
```
Then run scripts with
``` 
 PYTHON_PATH src/tasks/hello_world/hello_world.py
```
